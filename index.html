⚙ Settings & Export

🎫 Event Card Import

Paste a cagematch event card URL (ending in page=3) to import all matches from a show at once. Requires Apps Script URL below. Auto-detects TITLE CHANGE !!!

🔗 Individual Match Import

Paste one or more cagematch match URLs (one per line). Optionally add a rating and flags after a space:
URL 9.5 rewatch   URL 8 saw   URL 10 +   URL saw rewatch

🔄 Refresh CageMatch Data

Re-fetch CageMatch data for all matches in a year. Updates scores, winners, championships, and other fields. Your ratings, 10+, Saw Live, and Rewatch flags are preserved.

🔃 Sync

Merge local matches with your Google Sheet. Pulls new matches from the Sheet, then pushes any local-only matches back. Uses the same Apps Script URL above.

💾 JSON Backup

Full backup of all matches including flags and ratings. Use this to save or transfer your data between devices.

📊 CSV

Export for spreadsheet apps. Import merges — rows with an existing ID update in place, rows without an ID are added as new entries.

🔒 Password Protection

Generate a password hash, then paste it into the EMBEDDED_PW constant in the HTML file and push to GitHub.

⚡ Auto-Fill (CageMatch)

Auto-Fill and Event Card Import require a Google Apps Script Web App. Set the URL once — it handles both features.

⚠ Danger Zone

Delete all locally stored match data. Cannot be undone — download a JSON backup first.
