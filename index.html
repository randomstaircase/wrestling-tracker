Setup Instructions ►
  1. Open your Google Sheet → Extensions → Apps Script
  2. Replace all content in Code.gs with the provided WebApp_Code.gs file
  3. Click Deploy → New Deployment
  4. Type: Web App | Execute as: Me | Who has access: Anyone
  5. Click Deploy → copy the Web App URL → paste below
  6. Future script edits: Deploy → Manage Deployments → Edit → New version
